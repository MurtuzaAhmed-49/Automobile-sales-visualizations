## Automobile Sales Visualizations

Worked on historical automobile sales trends during **recession** and **non-recession** periods as part of a **Coursera** project and created visualizations using Python, **Matplotlib**, **Seaborn**, and **Folium**.  

### Features
- Line charts for sales trends across years.
- Bar charts for vehicle-wise sales comparison.
- Pie charts for advertising expenditure distribution.
- Scatter and bubble plots analyzing consumer confidence, unemployment rate, vehicle price, and seasonality effects.
- Interactive map using Folium to display sales across cities.
